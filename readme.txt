Landing Page Launchpad(LPL) Plugin Connector.

== Description ==

Landing Page Launchpad(LPL) Plugin helps users to add wordpress pages using LPL created pages by following few simple steps.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the LPL Settings Tab to configure the plugin
4. Use LPL Pages Tab to create Landing Page Launchpad pages in you wordpress.

== Arbitrary section ==

Landingpagelaunchpad (LPL) services
LPL is platform where user can create html pages from the list of templates that are provided by LPL site.
User can select any template, costimzed that template by adding content, images, links, optins. Further user can also add tracking code, seo tags and many other features. 
After saving user has the ability to download html, copy url and publish on wordpress.

Publish on wordpress
In this option LPL has built its own customized plugin.
The plugin required an api token to activate. The api token is provided by LPL site present in the account setting section. That api token is unique for each users.
User can copy that token and put in the setting options of the plugin to activate.
Afterward the plugin will start interaction with lpl api’s
LPL API
https://www.landingpagelaunchpad.com/api/lpl/lplpages
This is a public api that is built so that wordpress plugin can communicate with LPL site.
Api require api_key which user has got from LPL site.
